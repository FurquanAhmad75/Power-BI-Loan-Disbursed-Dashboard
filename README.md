# Power-BI-Loan-Disbursed-Dashboard
The Loan Disbursed Dashboard provides a comprehensive analysis of loan distribution, performance, and status. It helps stakeholders monitor total loan amounts, disbursement trends, approval status, and loan distribution across different categories.

# 🚀 Key Features
# 💰 Financial KPIs
Total Loan Amount: 1B
Total Disbursed Amount: 1B
Total Interest Rate (Aggregated): 9.40K
Total Loan Count: 1K

# 📈 Monthly Disbursement Trends
Visualizes disbursed loan amounts by month
Identifies seasonal patterns and fluctuations
Helps track peak and low-performing months

# 🏦 Loan Distribution Insights
Tenure by Loan Type
Displays loan duration distribution across categories
Helps understand customer borrowing behavior
Loan Status Breakdown
Active: 64.9%
Closed: 24.4%
Default: 10.7%
Enables quick assessment of portfolio health

# 🎯 Performance Tracking
Displays actual loan amount vs goal
Example:
Achieved: 110M+
Goal: 111M
Variance: -1.32%
Useful for tracking business targets

# 📋 Detailed Loan Records
Tabular view including:
Loan Type
Status
Loan Amount
Disbursed Amount
Loan Officer
Enables granular analysis and auditing

# 🛠️ Tools & Technologies
Power BI – Data visualization
DAX (Data Analysis Expressions) – Calculations & measures
Data Modeling – Relationship management

# 📊 Insights & Benefits
Monitor overall loan performance in real-time
Identify trends and risk (e.g., default loans)
Track disbursement efficiency
Support data-driven decision making
