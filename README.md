# Power-BI-Loan-Disbursed-Dashboard
The Loan Disbursed Dashboard provides a comprehensive analysis of loan distribution, performance, and status. It helps stakeholders monitor total loan amounts, disbursement trends, approval status, and loan distribution across different categories.

# 🚀 Key Features
# 💰 Financial KPIs
Total Loan Amount: 1B <br>
Total Disbursed Amount: 1B <br>
Total Interest Rate (Aggregated): 9.40K <br>
Total Loan Count: 1K <br>

# 📈 Monthly Disbursement Trends
Visualizes disbursed loan amounts by month <br>
Identifies seasonal patterns and fluctuations <br>
Helps track peak and low-performing months <br>

# 🏦 Loan Distribution Insights
Tenure by Loan Type <br>
Displays loan duration distribution across categories <br>
Helps understand customer borrowing behavior <br>
Loan Status Breakdown <br>
Active: 64.9% <br>
Closed: 24.4% <br>
Default: 10.7% <br>
Enables quick assessment of portfolio health <br>

# 🎯 Performance Tracking
Displays actual loan amount vs goal <br>
Example: <br>
Achieved: 110M+ <br>
Goal: 111M <br>
Variance: -1.32% <br>
Useful for tracking business targets <br>

# 📋 Detailed Loan Records
Tabular view including: <br>
Loan Type <br>
Status <br>
Loan Amount <br>
Disbursed Amount <br>
Loan Officer <br>
Enables granular analysis and auditing <br>

# 🛠️ Tools & Technologies
Power BI – Data visualization <br>
DAX (Data Analysis Expressions) – Calculations & measures <br>
Data Modeling – Relationship management <br>

# 📊 Insights & Benefits
Monitor overall loan performance in real-time
Identify trends and risk (e.g., default loans)
Track disbursement efficiency
Support data-driven decision making
